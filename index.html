The Project

For this project we had to compute a landslide susceptibility map and use it to evaluate the exposure of the population to landslides. Our group has been assigned an area east of Bergamo in the Valle Cavallina. This region includes Lake Endine and several neighboring towns.
Landslide susceptibility maps illustrate the likelihood of landslides occurring in specific areas, helping to highlight regions with higher spatial probability levels for potential landslides.
Exposure assessment involves identifying and quantifying the number of people or other elements located within areas at risk of hazards. This process is crucial for understanding the potential impact of natural disasters.

Dataset

  • Landslide inventory
    from IFFI catalogue with a scale of 1:10000
  • Digital Terrain Model
    from GeoPortale Lombardia with a spatial resolution of 5 m/px
  • Land use and land cover
    Destinazione d'Uso dei Suoli Agricoli e Forestali from GeoPortale Lombardia
  • Road and rivers network
    from OpenStreetMap
  • Distance to geological faults
    from GeoPortale Lombardia
  • Normalized Difference Vegetation Index
    derived from images of ESA Copernicus Sentinel-2

Tools used

QGIS is a geographic information system (GIS) software that is free and open-source. QGIS integrates with other open-source GIS packages, including PostGIS, GRASS GIS, SAGA GIS, and MapServer.

Plugins are software components that add a specific feature to an existing computer program. For QGIS they are usually written in Python or C++. We used Dzetsaka for semi-automatic classification with the Random Forest learnig method.

GeoServer is an open-source server written in Java that allows users to share, process and edit geospatial data. GeoServer implements several Open Geospatial Consortium protocols including Web Map Service (WMS) that we used to add layers to our gis.

OpenLayers is an open-source JavaScript library for displaying map data in web browsers as slippy maps. It provides an API for building rich web-based geographic applications similar to Google Maps and Bing Maps.

Vue.js is an open-source model-view-viewmodel front end JavaScript library for building user interfaces and single-page applications.

Visual Studio Code is a source-code editor developed by Microsoft for Windows, Linux, macOS and web browsers.

GitHub is a developer platform that allows developers to create, store, manage and share their code. It uses Git software, providing the distributed version control of Git
